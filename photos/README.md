## Recommended specs

- **Format:** JPG or PNG (JPG preferred for smaller size)
- **Aspect ratio:** 16:9 (landscape) — photos are cropped to this ratio via CSS
- **Resolution:** ~800–1200px wide is plenty
- **File size:** aim for under ~200 KB each (compress with TinyPNG or similar)
